# Riwayat Perubahan (Changelog) KITA SMP Landing Page

Semua rilis penting untuk proyek ini akan dicatat di file ini secara berkala.

## [1.0.0] - 2026-07-07

### Added
- **Desain Landing Page Premium:** Desain bertema *fantasy RPG* naga ungu, kristal mistik, dan gaya modern.
- **Header Responsif Sticky:** Navigasi otomatis mengunci di atas dengan efek blur transparan.
- **Dynamic Developer Fetch:** Integrasi API Github Raw untuk memuat metadata Ran Dev secara otomatis.
- **Copy-to-Clipboard IP Server:** Tombol salin satu ketukan dengan respon toast visual untuk Java & Bedrock.
- **Integrasi Skin Minecraft:** Tampilan grid tim staf menggunakan API Minotar sesuai skin asli in-game.
- **Kelengkapan SEO & Skema:** File `index.html` terstruktur penuh dengan data JSON-LD (Organization, WebSite, Breadcrumbs, FAQ).
- **Pengaturan Deployment Vercel:** Menyertakan `vercel.json` untuk pengoptimalan cache, gzip kompresi, dan perlindungan keamanan header.
